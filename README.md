# Studio 💾

The brand central for Karma Lies, powered by [`SvelteKit`](https://github.com/sveltejs/kit).

Built on the JAMStack.

## 🏗 Developing

Once you've created a project and installed dependencies with `pnpm i`, start a development server:

```bash
pnpm dev
```
